# Zomato_data_analysis

**The Zomato Data Analysis Project** Is a complete end-to-end data analytics case study designed to understand how restaurants perform on Zomato and what factors influence customer choices.
By analysing a real-world dataset, this project demonstrates how data-driven insights can help restaurants improve visibility, customer satisfaction, and business performance.
This project covers the complete analytics workflow:  
- Loading a dataset in Python  
- Performing EDA  
- Cleaning data  
- Running SQL queries  
- Building a Power BI dashboard to present key insights  

---

## Dataset Source  
Zomato restaurant dataset (CSV) containing details such as:
- order_id
- customer_id
- restaurant_name
- cuisine
- order_datetime
- delivery_time_min
- order_value
- discount_percent
- payment_method
- rating
- delivery_city etc

---

## Tools Used  
- Python  
- SQL  
- Power BI  
- Jupyter Notebook  

---

## Steps  
1. Loaded dataset in Python  
2. Performed EDA and cleaned the data  
3. Wrote SQL queries for deeper analysis  
4. Built an interactive Power BI dashboard to visualize insights  

---

## Dashboard  
Interactive Power BI dashboard showing key metrics like:  
- Total_revenue
- Total_order
- Rating of Restaurant
- Avg delivery time etc  

---

## Results  
- Clean and structured Zomato restaurant dataset
- Insights into customer preferences, restaurant categories, and rating patterns
- Clear data visualizations through Power BI for easy interpretation

---

## How to Run  
- Open the **.ipynb** notebook in Jupyter  
- Install required libraries using `requirements.txt`  
- Run SQL queries in your database client  
- Open the **.pbix** file in Power BI Desktop to view the dashboard
