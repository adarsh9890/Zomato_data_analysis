# Zomato_data_analysis

Data analytics project focusing on restaurant and food-industry insights using Python, SQL, and Power BI.
This project covers the complete analytics workflow:

Loading the Zomato dataset in Python

Performing EDA and data cleaning

Running SQL queries for deeper exploration

Building a Power BI dashboard to highlight key findings

Dataset Source

Zomato restaurant dataset (CSV) containing details such as:

Restaurant name

Ratings & votes

Online ordering

Table booking

Approx cost for two

Restaurant type

Used for analysis, preprocessing, and visualization.

Tools Used

Python

SQL

Power BI

Jupyter Notebook

Steps

Loaded the Zomato dataset in Python

Performed EDA and cleaned the data (handling nulls, duplicates, inconsistent values)

Wrote SQL queries to analyze restaurant trends, ratings, costs, and preferences

Built a Power BI dashboard to visualize major insights

Dashboard

Interactive Power BI dashboard showcasing metrics such as:

Ratings distribution

Restaurant types

Online order vs offline performance

Cost for two analysis

Votes & popularity trends

Results

Clean and structured Zomato restaurant dataset

Insights into customer preferences, restaurant categories, and rating patterns

Clear data visualizations through Power BI for easy interpretation

How to Run

Open the .ipynb notebook in Jupyter

Install required Python libraries using requirements.txt

Run SQL queries using your preferred SQL client

Open the .pbix file in Power BI Desktop to view the dashboard
